一加是一种生活方式

发表于 2015-07-28 | |

     一直都很喜欢做工很精致的东西。
     生活用品要实用好用，这当然是一种基础，我也会渐渐提高对自己使用的东西的要求，它们也应该是符合自己心态的，作为一种可以体现本身生活态度的存在。
对于衣物家居，以纯、优衣库、无印良品、JOYSTYLE、梵几都是符合我的审美的品牌，简洁自然的风格，往往可以让我很快投入到自己所的事情中去。
     关注OnePlus这个品牌也有一段时间了，它给人的感觉似乎是一家日本的企业，在YouTube上看了许多关于OnePlus one的测评，好用、实用、有格调这是给我的感觉。
     于是，一直都在等待其第二代的产品，7月28日看完了全程OnePlus two的发布会，难能可贵的是没有感觉到其它国产手机发布会上的自说自话。当然，还好OnePlus没有做其它的产品，现在它专注于手机也就够了，手机看起来足够好用，系统也是我喜欢的清新风格。
当听到“最通透的屏幕”这样的表述时，便有了一种旧友重逢的感觉，这不就是我一直对手机屏幕的追求吗？
     当自己的生活被智能手机改善的同时，我也要忍受他们对我独处时光的侵犯，现在自己已经关闭了许多APP的推送功能，我没有必要第一时间就去知道这个世界突然发生了什么，更何况那报道还是各种让人难以接受的词语组合起来的。
     科技对于生活的改善应该是低频的，处于一种存在感很小的状态。

     英剧 黑镜 就向我们展示了一个科技全面入侵私人生活的未来社会，这是灾难性的。或者说现在的社交网络都是讲求快速、短洁，这就必然导致内容质量的低下。
     追求安静的生活，所以选择了OnePlus Two作为自己下一步手机。
