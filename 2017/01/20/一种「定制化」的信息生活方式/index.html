一种「定制化」的信息生活方式

发表于 2017-01-20 | 分类于 生活 | |

「本文参与 少数派2016年度征文」

摘要：本文主要介绍我在这个信息爆炸时代「定制化」生活的方法和些许思考，获取真实的信息，建立合适的流程，深化对知识的理解，输出不那么浅薄的思考。「定制化」就是对生活做减法，拒绝无用信息的轰炸，拒绝低效的工作流程，拒绝独立思考空间的沦陷。

前述

大学伊始，我还有着每日一记的好习惯，深夜11点我会在书桌前安静写下一天的心得和思考，有时思路和感想来了，不知不觉几百字的小文章竟已经写下，和自己交流整个人都会沉静下来。后来，智能手机进入我的生活，我享受着移动互联网发展带来的各种便捷。渐渐地，便将自己的生活交付给了它们，它们管理我的日程安排、信息、支付、购物、交通……这实在是最好的障眼法：各种推送和短信关怀，各种读取权限的给予，让我便捷，也让我疯狂。恍惚间，日记没写了，账单流水没了头绪，沉迷于各种点赞，迷失在各种嘈杂的信息中。不行，我得反击，夺回生活的主动权。战斗开始了，我尽力抓住生活的脉络，把握事物的节奏，避免在手机屏幕上无意识的点击。

我得知道这个信息社会是如何一步步在不知不觉中让我放弃了生活的自主权，拱手让人。

了解信息社会

我们这一代最杰出的头脑都在拼命思考如何吸引人点击更多的广告。

这句话出自生于1983年的数学天才Jeff Hammerbacher，他本人曾经效力的Facebook所依赖的商业模式正是“拼命思考如何吸引人点击更多的广告”。

国内的BAT又何尝不是把这些发挥到了极致呢？结合了人性考量的产品设计轻易偷走你的注意力，基于大数据的个性化推送和各种关怀方式让你时刻在虚无的成就感中获得满足。微信朋友圈的小红点、微博的粉丝数、知乎回答的赞同数，一个又一个的量化指标不断引发你的颅内多巴胺的分泌，给以幸福的幻觉。然而，不断攀高的量化指标对现实生活的改变呢？当人类这一代最杰出的头脑拼命思考如何控制我们的生活的时候，难道我们只能束手就擒吗？

人类历史已经进入信息时代，拒绝它显然是不明智的。那就融入它，了解它，分析它，利用它，战胜它，为我所用吧！开始定制自己的生活吧！「定制化」就是对生活做减法，拒绝无用信息的轰炸，拒绝低效的工作流程，拒绝独立思考空间的沦陷。

筛选信息源

拒绝无用信息的轰炸，我选择的是RSS、即刻、播客三样工具。

RSS

使用RSS——Really SimpleSyndication（真正简易联合），让你可以把控信息的输入，而不是被动接收信息。它的主要作用如下：

  • 有选择地浏览您感兴趣的以及与您的工作相关的新闻；
  • 把需要的信息从不需要的信息（兜售信息，垃圾邮件等）中分离出来；
  • 创建自己的新闻频道，并将之发布到因特网。

使用RSS feed

使用RSS feed分为手机端和电脑端，适用于不同的场景。

  • 手机端

经过对不同软件的比较，建议使用Inoreader的服务，稳定可靠且无需FQ便可以查看订阅的境外信息源。另外，feedly也可选，智能订阅简单易用。

  • PC端

Inoreader网页版设计简约，逻辑清晰，并且可以根据已订阅的内容智能推荐新的订阅源，帮你探索新的世界。

偶尔也会需要使用RSS桌面客户端，推荐且只推荐FeedDemon，这是一个稳定可靠的RSS订阅工具，便于整理收藏修改信息。

优质订阅源推荐

  • 知乎每日精选（强烈推荐）：每天推送知乎优质内容；
  • 果壳网（科普生活）：果壳网是一个泛科技主题网站，会推送许多有趣好玩但不失严谨的知识；
  • Matrix67（数学爱好者）：研究有趣的数学知识；
  • 战隼的学习探索（效率生活）：提供了很多经过验证的提高生活和学习效率的方法；
  • 联合早报（客观新闻）：新加坡、中国、亚洲和国际的即时、评论、商业、体育、生活、科技与多媒体新闻，尽在联合早报；
  • 译言精选（国外趣闻）：中国最大的译者社区和众包翻译平台，可以申请分包翻译任务哦；
  • 南都周刊（犀利观点）：中国唯一一份以城市为关注对象的新闻杂志；
  • 异次元软件世界（有态度的软件）：一个相当低调的网站，分享各种精选优质的软件、游戏、电影等资源。

对RSS的思考

看到大家都推荐使用RSS，但是很多都是订阅了很多信息源，几乎没有时间和精力看完，少订阅几个信息，多一些自己的思考，及时复盘，偶尔逼迫自己输出，这样子就可以最大化自己的创作效率。说到底，我们获取有效的信息不正是为了高质量的输出吗？输出才是关键，你说呢？！

即刻

「即刻」可以说是2016年给我最大的惊喜的App，这是一款基于兴趣的极简信息推送工具，你可以通过它关注自己感兴趣的人物、资讯和事件，它会通过推送让你轻松、高效、及时地获取自己关心的优质信息。

即刻让我爱上了推送，不再需要进行繁琐的推送权限设置便可以获取繁复的互联网世界关注的信息。

但是最近我也发现了一个问题，即刻有时候给我一种已经和世界紧密联系的幻觉，它的信息有趣但有限，还有好多领域是并不有趣但值得我们去关注和了解的，测绘、医疗、半导体产业链等等这些信息我还是需要通过阅读文献才可窥得一斑。

播客

声音是文字之外最重要的信息来源，睡前听下的声音第二天还会在脑袋中回响，选择优秀的订阅工具和内容就很重要了。我使用的是CastBox，纯正的material design设计，充足的中文内容源，优秀的交互体验，又一次将我从诸如喜马拉雅这样的客户端中拯救出来。

优质播客推荐

中文播客

罗辑思维：如果你是喜欢独立思考的人物，这个播客一定已经听过了。

「I/O」调频：作者有自己博客，文章质量很优质，播客的内容也很好，刚开始不久，值得入坑

外文播客

一点思考

拒绝了无用信息的轰炸，那我该如何获取有效真实且有深度的信息呢？

深感此处别无它法，互联网社会的信息利于快速传播就很难兼顾深度，唯有阅读专业文献。对于大数据和人工智能的报道看了那么多，也没有一个稳固的概念，在仔细阅读美国白宫大数据白皮书并做下详细笔记之后才敢说自己了解这些东西。

当然，到了具体的行业大数据该如何运用和实现又是一个巨大的课题。有几家咨询机构出的报告还满专业和可靠的，可以去瞅瞅。

搭建工作流

拒绝低效的工作流程，我选择的是拥抱Google、幕布、treation、Leangoo、Axure、BitTorrent Sync。

如何使用Google似乎是持久的话题，人人都在寻找练成气功飞跃的办法，这里就说说两种正常使用的方案和我的Google工作法。

Host让人烦恼的地方就是经常需要修改才可以正常使用，那有没有可以自动更新host的办法呢？这就需要Hozz这个工具了，看看它的介绍：

  • Automatic update of online hosts
  • Import hosts file by dragging them into the sidebar
  • Export and import hosts files
  • Support hosts sorting through dragging

这不正是我们需要的吗？自动升级在线的hosts。

有了这个工具我们只需要好好每天打开自动更新一下就好了。Android手机上root后也是可以修改host的，那有没有可以自动更新的工具好用呢？当然也是有的。

Google Calendar

这大概是最好的日历应用吧，可以和滴答清单连接，同步日程，也可以订阅自己关注的体育比赛、影视动态、名人近况等，结合着Google feed，不会让我错过任何一件记录下的事情。

Google keep

这个多端同步的便签扮演着购物清单、生活账单、会议纪要、电费清单记录者的角色，支持清单和录音，许多会议的录音就是这么记录下来的。

各种突然闪现的想法我都会用这个记下，然后用幕布进行深度的思维归纳。

幕布

2016年最让我难忘的思维概要整理工具，之前使用workflowy，实在是难以习惯。邂逅幕布，一下子就解决了自己使用思维导图的尴尬症。这不正是我们需要的那种使用方便、设计简洁的产品吗？它可以用来写笔记、整理思维、内容创作、任务管理，快速整理出头脑中的内容大纲，再去细化每一个片段，使内容的结构更加清晰。

Treation

Treation是一个无比灵活的在线表格，你可以用它和人任何人管理任何事情。

这是Treation的理念，让大家友好灵活地在线办公，这恰恰解决了我所在团队的痛点，再也不需要每日邮件发送工作日报啦。原本需要手工完成的事情，在 Treation 里只要轻松添加一些规则就可以将它们自动化；表格的每一列都可以设置为不同的类型，无论图片、文件、日期、选项，还是公式、自动完成，都可以很方便的进行管理和使用。

春节的一些同学聚会、年货购置、家电维修的事情我也尝试放在这上面来规划，结合Leangoo使用往往可以起到很好的效果。

Axure

Auxre大概是产品经理的必备软件吧，这个软件不仅可以画原型图、流程图，还可以用于绘制房屋结构图、文字排版、P图等，可以说是一个难得的利器。之前房屋转租的信息发布在豆瓣上附带了使用axure制作的房屋平面图，很快就顺利转租出去了。

BitTorrent Sync

BitTorrent Sync是由BitTorrent公司开发的专有的对等网络数据同步工具，可在Windows、OS X、Linux、Android、iOS和FreeBSD上使用。其可在局域网、互联网上通过安全的、分布式的P2P技术在不同设备之间同步文件。

（引用自维基百科）

Sync具有稳定、免费、高效的特点，它是一款神奇的文件同步软件，它无需服务器，便可让多台电脑间无限制互相同步！但更神奇的是，它可以作为下载软件使用，只要你有Key！同时你也可以将自己的文件同步，在不同终端实现文件的稳定同步，甚至可以团队内的协同办公。

显然，key才是使用的关键，互联网世界就是一个金矿，等待着你去发现，去寻找或者创建属于你的key吧！

这是2016年双十一4899元购入的，起初并没有很高的期待，只要可以流畅使用就OK了，实际的使用体验是优于市面很多同价位的超极本的，外形硬朗，系统纯粹，256GB的SSD加持也是轻松应对各种软件。当然，游戏党和性能党不必考虑。Win10平台上也是有诸如网易云音乐、Todoist、NEBO笔记软件等诸多优秀UWP软件。

iPad mini4

iPad mini4是绝佳的阅读器和游戏工具。前前后后购买了饥荒、Crashlands等游戏，iBooks的阅读体验很好。Apple家的产品体验确实很棒，奈何我是忠实的Google党~

kindle

kindle对于阅读爱好者是一大利器，半夜里kindle为伴，似乎可以忘记了时间，古人凿壁偷光，今者电子墨水漫反射。亚马逊家的kindle unlimited包年服务也是很厚道，不过kindle体验再好，我也改不了买本纸质书享受随意标注的坏习惯。

作结

在信息化社会中过着「定制化」的生活其实是并不是一件省心的事情，但是这种选择是一种积极的生活态度，需要自己的调教和取舍，静静诉说着自己在这个喧嚣社会的坚守，就像钱钟书写的那样：

快乐时的你一定心无愧怍。

谢谢大爷,您嘞~~
