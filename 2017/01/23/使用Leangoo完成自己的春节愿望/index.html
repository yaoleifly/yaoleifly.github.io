使用Leangoo完成自己的春节愿望

发表于 2017-01-23 | 分类于 生活 | |

踩不住公历2016年的尾巴，总要给个机会完成农历春节的愿望清单吧~

爆竹声中一岁除，春风送暖入屠苏。千门万户曈曈日，总把新桃换旧符。

这「新桃换旧符」不会凭空出来，活这么大，也没有遇到几件无端端自己就发生的好事儿，自己总得做些事情吧~

凡谋之道，周密为宝。

——《六韬》

那就来做个周密的计划吧。提到计划，可以用的工具实在太多，滴答清单、微软家的奇妙清单，Google家的日历或者Keep，不一而足。但是，要是提到周密计划，这些看起来轻量的工具有时候也是不堪大用，求助于人，不如回顾过往经验。大型规划？周密细节？多人协作？那不正是项目管理的要求吗？

这里就选Leangoo这款简洁、轻量、可视化的项目管理工具来做我的春节愿望清单吧~

说干就干！

关于leangoo

Leangoo是一款非常简洁的团队的协作工具，它的设计理念是围绕看板进行团队协作,设计融入了很多敏捷思想，功能上也加入了大量支持敏捷的元素，比如泳道、燃尽图、敏捷工作量估算等。（好复杂呀~好多功能我都不用的。）

规划清单

登录Leangoo注册账号，绑定微信可以直接手机查看。

进入之后就是这个界面了，我已经把里面官方提供的示例都归档了，免得影响我使用。

看到我的看板2017春节愿望清单没有？

春节要做的四件需要规划的事情如下：

  • 承接2个微信小程序外包业务
  • 为女友准备新年礼物
  • 维修热水器
  • 维修台式机

一件一件来规划吧~

微信小程序外包

我的家乡某city隶属于湖北省，由孝感市代管，位于鄂中腹地，是楚文化发祥地，是历史上郧子国、德安府所在地。截止2012年，土地面积1355平方公里，人口63万，辖9镇4乡2个办事处1个经济技术开发区。某city存有许多经营良好的商铺，其中烧烤外卖、美容美发预约、酒店订座、婚纱摄影预定、KTV等等可能存在着小程序设计的需求，有待发掘。

为此提出建立某city微信小程序外包团队，为有需求的店铺提供一体化的小程序构建服务，优化商家服务质量和拓宽服务途径。
