「创新型」宣教对住院患者满意度提升的效果评价

发表于 2017-04-07 | 分类于 工作 | |

一、调研概述

1.1调研目的

研究「创新型」宣教对省儿保呼吸内科和五官科住院患者或家属的住院满意度的提升效果。

1.2调研背景

患者满意度是评估医院医疗服务质量和服务水平的重要参数。通过对患者满意度进行评估与分析，可以发现患者对于医院服务的期望与事实感受之间的差距，了解医院服务的薄弱环节及其影响因素，从而有针对性的为医院提高服务质量、改善患者就医体验、构建和谐医患关系等提供合理化建议，进一步提升医疗服务水平。

二、调研方法

选取浙江省儿童医院滨江院区的住院患者或其家属作为研究对象，根据宣教方式的不同，随机分为实验组（n=200）和对照组（n=100）。对照组患者或家属接受传统宣教，实验组患者在传统宣教的基础上进行创新型宣教。比较两组患者或家属住院期间宣教满意度、护理满意度以及整体满意度的情况。以实际取得的样本数量为准。

对照组患者或家属进行传统宣教（口头和纸面），实验组患者或家属在传统宣教的基础上进行创新性宣教（317护）。

对照组的调研工作由员祎 负责，在医院出入院办理处 、住院协调处、住院部15楼五官科 进行。

2.1出入院的创新型宣教

出入院环节的创新性宣教包括 入院前或入院时微信获取入院宣教课程 和 出院时或出院后微信获取出院宣教课程 。由于出入院宣教的特殊性，患者或家属及时收到宣教课程，可以帮助患者更好地办理出入院手续、医保手续、缴费账单等，提高患者认知，降低护士工作量，提升服务质量。因此，调研人员会在出入院办理处放置课程二维码和优化好的课程手册，供患者获取并阅读。观察已经获取（主动获取或引导获取）并阅读课程的患者或家属，与其沟通，表明调研身份，让其填写满意度调查问卷，获取样本。

该部分工作由 姚磊 负责，在医院 出入院办理处 和 住院协调处 进行。

2.2住院部的创新型宣教

住院环节的创新性宣教包括 住院期间微信推送 疾病类宣教、检查类宣教、流程类宣教。由于住院期间患者或家属会遇到疾病、饮食、手术前后、心理、康复等多样化的宣教需求，患者或家属可以收取到多样化的宣教课程，可以帮助其更好地了解健康知识，减少医患的知识不对称，促进医患和谐。同时，由于住院宣教的随机性，难以持续捕捉护士宣教的时机，故采用以下两种方法选择目标对象：

  1. 从课程后台选取已读课程的患者，与其沟通，表明调研身份，让其填写满意度调查问卷，获取样本；
  2. 在电梯等待处、楼道休息处随机选择，与其沟通，表明调研身份，让其阅读课后填写满意度调查问卷，获取样本。

该部分工作由 姚磊 负责，在医院 住院部20楼21楼呼吸内科 进行。

2.3调研地点

  • 省儿保住院部（实验组和对照组）
  • 15楼 耳鼻咽喉头颈外科 眼科 口腔科 （对照组）
  • 20楼 呼吸内科1（实验组）
  • 21楼 呼吸内科2（实验组）
  • 出入院办理处（实验组和对照组）
  • 床位协调处（实验组和对照组）

2.4观察指标

比较两组患者出入院、住院期间的宣教满意度和护理满意度的情况。

2.5统计学处理

采用 SPSS 22.0 统计学软件对数据进行分析，计数采用 % 表示，进行 χ2 检验。

三、问卷设计

3.1指标体系

规定了医院住院患者满意度评估指标体系的设置原则和基本构成。

调研住院患者对医院宣教的满意度，二级指标包括 5个指标：

  1. 口头宣教
  2. 纸面处方宣教
  3. 电视视频宣教
  4. 宣传栏宣教
  5. 信息化宣教（317护）

用于测量住院患者对医院现有5种宣教服务形式的满意情况。

三级指标为内容、形式、时机满意度3个维度：

内容满意度3个：

  1. 针对性
  2. 逻辑性
  3. 通俗性

形式满意度4个：

  1. 图文协调
  2. 风格统一
  3. 排版合适
  4. 篇幅适宜

时机满意度2个：

  1. 入院前或入院后可以获取
  2. 住院时及时获取宣教课程

（具体数目待定）

四、调研结论

「创新型」宣教具有良好的宣教效果，可以减少护士60%的宣教工作，提高患者的满意度和健康知识掌握水平，具有显著的患者满意度提升效果，对住院患者的就医、用药、饮食、心理、康复具有重要价值，值得在医院推广。
